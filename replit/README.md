# FORGE on Replit — Try It Instantly

## How to Use
1. Click "Fork" or "Run" to create your own copy
2. Go to Secrets tab (lock icon in sidebar)
3. Add: `GEMINI_API_KEY` = your free key from [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
4. Open the Shell tab
5. Type: `forge-cli`

That's it — no install, no Rust toolchain, no nothing.
